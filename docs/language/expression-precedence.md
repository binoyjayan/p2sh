# Expression precedence

| Operators / Expression     | Associativity |
|----------------------------|---------------|
| [] . ()                    | left to right |
| ! - ~ (Unary)              | left to right |
| * / %                      | left to right |
| + -                        | left to right |
| << >>                      | left to right |
| &                          | left to right |
| ^                          | left to right |
| \|                         | left to right |
| == != < > <= >=            | left to right |
| &&                         | left to right |
| \|\|                       | left to right |
| .. ..=                     | left to right |
| \| (in match pattern)      | left to right |
| =                          | right to left |
